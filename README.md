## BIOS设置
- BIOS版本089
 ### Disable
 - Legacy Boot
- Fast Boot
- Network Boot
- Secure Boot
- Inter VT for directed I/VO(VT-d)
### Enable
- Boot USB Devices First
- Boot Network Devices Last
- Wake on LAN from S4/S5
- Stay Off
